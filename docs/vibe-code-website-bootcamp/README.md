# Vibe Code Website Bootcamp - Teaching & Student Materials

ชุดเอกสารนี้แตกเนื้อหาจาก `vibe_code_website_bootcamp_summary.md` ออกเป็น 2 ชุด:

1. **Instructor Pack** - ไฟล์ระดับบน ใช้สำหรับเตรียมสอน คุมลำดับคลาส และตรวจงาน
2. **Student Pack** - ไฟล์ในโฟลเดอร์ `student/` ใช้แจกผู้เรียน อ่านตาม และกรอกระหว่าง Workshop

แนวคิดหลักของคอร์ส:

> เริ่มจากโจทย์ให้ชัดก่อน แล้วค่อยใช้ AI ช่วยออกแบบ เขียนโค้ด แก้บั๊ก Deploy และวัดผล

## Instructor Pack

1. `00-course-overview.md` - ภาพรวมคอร์ส เป้าหมาย ผู้เรียน ผลลัพธ์ และตารางเวลา
2. `01-vibe-code-foundation.md` - Day 1 Session 1: Vibe Code, Project Brief, การเลือกโจทย์
3. `02-landing-page-design-html-preview.md` - Day 1 Session 2: Section, UX/UI, HTML Preview
4. `03-nextjs-build-workflow.md` - Day 2 Session 3: Next.js, Component, GitHub, Debug
5. `04-deploy-tracking-portfolio.md` - Day 2 Session 4: Form, Supabase, Vercel, Tracking, Portfolio
6. `05-current-stack-and-tools.md` - Stack และเครื่องมือที่ควรรู้สำหรับ Web / Mobile / AI Coding
7. `06-ai-coding-skills-context.md` - Skill, Prompt Workflow และ Context Management
8. `07-prompt-pack.md` - Prompt Pack สำหรับผู้เรียน
9. `08-workshop-checklists.md` - Checklist รวมสำหรับ Workshop และก่อนจบคอร์ส
10. `09-student-templates.md` - Template ให้ผู้เรียนกรอกระหว่างเรียน

ไฟล์ชุดนี้ยังใช้ภาษาฝั่งผู้สอนในหลายจุด เช่น "บทนี้ใช้สอนเพื่ออะไร", "สิ่งที่ควรย้ำ", "Instructor Quick Review" จึงเหมาะกับการเตรียมคลาสมากกว่าการแจกผู้เรียนตรง ๆ

## Student Pack

ไฟล์สำหรับแจกผู้เรียนอยู่ใน `student/`

1. `student/README.md` - คู่มือเริ่มต้นสำหรับผู้เรียน
2. `student/full-student-handbook.md` - คู่มือผู้เรียนฉบับละเอียด โครงใกล้กับเอกสารผู้สอน แต่ใช้ภาษาผู้เรียน
3. `student/01-learning-path.md` - เส้นทางการเรียนและสิ่งที่ต้องส่งแต่ละช่วง
4. `student/02-workshop-guide.md` - ขั้นตอนทำ Workshop ตั้งแต่ Brief ถึง Deploy
5. `student/03-prompt-pack-for-students.md` - Prompt ที่คัดให้ผู้เรียนใช้ทันที
6. `student/04-student-worksheet.md` - ใบงานสำหรับกรอกระหว่างเรียน
7. `student/05-final-submission-checklist.md` - Checklist ก่อนส่งงานท้ายคอร์ส

ไฟล์ `full-student-handbook.md` คือเอกสารหลักสำหรับแจกผู้เรียน ส่วนไฟล์อื่นเป็นเอกสารช่วยทำ Workshop, Prompt, Worksheet และ Checklist

## วิธีใช้เอกสารชุดนี้

สำหรับผู้สอน:

- อ่าน `00-course-overview.md` เพื่อจับภาพรวมและเวลาเรียน
- ใช้ไฟล์ `01` ถึง `04` เป็นแกนการสอนสด
- ใช้ `07-prompt-pack.md` เป็น prompt แจกผู้เรียน
- ใช้ `08-workshop-checklists.md` ตรวจผลลัพธ์ท้ายแต่ละช่วง
- ใช้ `09-student-templates.md` เป็นใบงาน

สำหรับผู้เรียน:

- เริ่มที่ `student/README.md`
- อ่าน `student/full-student-handbook.md` เป็นคู่มือหลัก
- อ่าน `student/01-learning-path.md` เพื่อเห็นภาพรวมแบบสั้น
- ทำตาม `student/02-workshop-guide.md`
- ใช้ prompt จาก `student/03-prompt-pack-for-students.md`
- กรอกใบงานใน `student/04-student-worksheet.md`
- ตรวจงานท้ายคอร์สด้วย `student/05-final-submission-checklist.md`
- เก็บผลลัพธ์สุดท้ายเป็น Portfolio หรือ Landing Page ใช้งานจริง

## ผลลัพธ์ปลายทางของผู้เรียน

หลังเรียนจบ ผู้เรียนควรมี:

- Project Brief ของเว็บไซต์ตัวเอง
- โครง Section ของ Landing Page
- HTML Preview 1 หน้า
- โปรเจกต์ Next.js ที่แยก Component
- GitHub Repository
- เว็บไซต์ออนไลน์บน Vercel
- Contact Form หรือ Lead Form เบื้องต้น
- Tracking Checklist
- Roadmap การต่อยอดเว็บหลังเรียน
