# 03 - Prompt Pack สำหรับผู้เรียน

คัดลอก Prompt เหล่านี้ไปใช้กับ AI ได้เลย แล้วแทนที่ข้อความในวงเล็บด้วยข้อมูลของคุณ

## Prompt 1: ช่วยเลือกไอเดียเว็บไซต์

```text
ฉันอยากทำเว็บไซต์สำหรับ [ธุรกิจ/บริการ/ตัวเอง]
ช่วยเสนอไอเดียเว็บไซต์ 5 แบบที่เหมาะกับฉัน

ข้อมูลของฉัน:
- สิ่งที่ทำ:
- กลุ่มเป้าหมาย:
- เป้าหมายที่อยากได้จากเว็บ:
- สิ่งที่อยากให้ผู้ใช้ทำ:

ขอให้เสนอแต่ละไอเดียพร้อม:
1. ประเภทเว็บไซต์
2. กลุ่มเป้าหมาย
3. เป้าหมายของเว็บ
4. CTA หลัก
5. เหตุผลที่เหมาะกับฉัน
```

## Prompt 2: สร้าง Project Brief

```text
ช่วยสร้าง Project Brief สำหรับเว็บไซต์จากข้อมูลนี้

ประเภทเว็บไซต์:
เป้าหมายของเว็บไซต์:
กลุ่มเป้าหมาย:
ปัญหาของกลุ่มเป้าหมาย:
จุดขาย:
Offer:
Tone & Mood:
สีที่ต้องการ:
CTA หลัก:
CTA รอง:
ข้อมูลที่ต้องการให้ลูกค้าทำ:

ช่วยจัดเป็น Project Brief ที่อ่านง่าย และเหมาะสำหรับนำไปทำ Landing Page
ถ้ามีข้อมูลส่วนไหนยังไม่ชัด ช่วยแนะนำว่าควรเติมอะไร
```

## Prompt 3: วางโครงหน้าเว็บ

```text
จาก Project Brief ด้านล่าง ช่วยวางโครง Landing Page ให้ฉัน

ขอให้จัด:
1. ลำดับ Section จากบนลงล่าง
2. เป้าหมายของแต่ละ Section
3. Headline ของแต่ละ Section
4. ข้อความอธิบายสั้น ๆ
5. CTA ที่เหมาะสม
6. สิ่งที่ควรระวังบนมือถือ

Project Brief:
[วาง Project Brief]
```

## Prompt 4: เขียนข้อความหน้าเว็บ

```text
ช่วยเขียนข้อความสำหรับ Landing Page จาก Section นี้

กลุ่มเป้าหมาย:
[ระบุกลุ่มเป้าหมาย]

น้ำเสียง:
น่าเชื่อถือ เป็นมิตร อ่านง่าย ไม่ขายแข็งเกินไป

Section ที่ต้องการ:
[วาง Section]

ขอให้เขียน:
1. Headline
2. Subheadline หรือคำอธิบาย
3. Bullet สำคัญ
4. CTA
5. ตัวเลือกข้อความอีก 2 แบบสำหรับเปรียบเทียบ
```

## Prompt 5: สร้าง HTML Preview

```text
ช่วยสร้าง HTML Preview สำหรับ Landing Page จากข้อมูลด้านล่าง

เงื่อนไข:
- ใช้ HTML + CSS ในไฟล์เดียว
- ดีไซน์ Modern, Clean, Premium
- รองรับมือถือ
- ใช้ภาษาไทย
- CTA ชัดเจน
- ไม่ต้องเชื่อมระบบจริง
- เหมาะสำหรับนำไปแปลงเป็น Next.js ต่อ

ข้อมูลหน้าเว็บ:
[วาง Section และข้อความ]
```

## Prompt 6: รีวิวหน้าเว็บ

```text
ช่วยรีวิว Landing Page นี้ในมุมผู้ใช้จริง

ช่วยดูว่า:
1. เข้าใจไหมว่าเว็บนี้เสนออะไร
2. กลุ่มเป้าหมายชัดไหม
3. Hero ชัดใน 5 วินาทีไหม
4. CTA เด่นไหม
5. Section เรียงดีไหม
6. มีจุดไหนที่ทำให้ไม่มั่นใจไหม
7. ควรแก้อะไรก่อน 5 อันดับแรก

ข้อมูลหน้าเว็บ:
[วางเนื้อหาหรือ HTML]
```

## Prompt 7: แปลง HTML เป็น Next.js

```text
ช่วยแปลง HTML Preview นี้เป็น Next.js App Router

เงื่อนไข:
- ใช้ TypeScript
- ใช้ Tailwind CSS
- แยก Component เป็นไฟล์ในโฟลเดอร์ components
- หน้าแรกอยู่ที่ app/page.tsx
- รักษา Layout, สี และข้อความเดิมให้ใกล้เคียงที่สุด
- โค้ดต้องอ่านง่ายและแก้ต่อได้

HTML Preview:
[วางโค้ด HTML]
```

