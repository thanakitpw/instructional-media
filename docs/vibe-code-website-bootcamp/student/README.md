# Vibe Code Website Bootcamp - เอกสารสำหรับผู้เรียน

เอกสารชุดนี้ใช้สำหรับเรียนและทำ Workshop ตลอดคอร์ส เป้าหมายคือให้คุณสร้างเว็บไซต์ของตัวเองได้จริง 1 เว็บ ตั้งแต่คิดโจทย์ วางโครงหน้าเว็บ ทำตัวอย่าง HTML แปลงเป็น Next.js และนำขึ้นออนไลน์

คุณไม่จำเป็นต้องเขียนโค้ดเก่งตั้งแต่เริ่ม แต่ต้องทำตามขั้นตอน ทดลองด้วยตัวเอง และใช้ AI เป็นผู้ช่วย ไม่ใช่ปล่อยให้ AI ตัดสินใจแทนทั้งหมด

## เริ่มจากไฟล์นี้

อ่าน [full-student-handbook.md](/Users/thanakitchaithong/Developer/projects/bestskill/instructional-media/API%20Integration%20for%20Web%20%26%20Mobile%20App/docs/vibe-code-website-bootcamp/student/full-student-handbook.md) เป็นคู่มือหลัก

ไฟล์นี้เป็นคู่มือผู้เรียนแบบละเอียด ใช้เป็นเนื้อหาหลักสำหรับอ่านตามระหว่างเรียนและทบทวนหลังจบคอร์ส

## สิ่งที่คุณจะได้ทำในคอร์สนี้

1. เลือกไอเดียเว็บไซต์ของตัวเอง
2. เขียน Project Brief ให้ชัด
3. วางโครง Landing Page
4. ใช้ AI ช่วยเขียนข้อความและออกแบบ Section
5. สร้าง HTML Preview
6. แปลงเว็บเป็น Next.js
7. แยกส่วนหน้าเว็บเป็น Component
8. ใช้ GitHub เก็บโปรเจกต์
9. Deploy เว็บขึ้น Vercel
10. วางแผนวัดผล เช่น click LINE หรือ submit form

## ไฟล์ในชุดผู้เรียน

- `full-student-handbook.md` - คู่มือผู้เรียนฉบับละเอียด ใช้เป็นเอกสารหลัก
- `01-learning-path.md` - เส้นทางการเรียนและสิ่งที่ต้องส่งแต่ละช่วง
- `02-workshop-guide.md` - ขั้นตอนทำ Workshop ตั้งแต่ Brief ถึง Deploy
- `03-prompt-pack-for-students.md` - Prompt ที่คัดมาให้ใช้ระหว่างเรียน
- `04-student-worksheet.md` - ใบงานสำหรับกรอก Brief, Section, Checklist และ Roadmap
- `05-final-submission-checklist.md` - Checklist ก่อนส่งงานท้ายคอร์ส

## ผลลัพธ์สุดท้ายที่ควรมี

เมื่อจบคอร์ส คุณควรมี:

- Project Brief ของเว็บไซต์ตัวเอง
- โครงหน้า Landing Page
- HTML Preview
- โปรเจกต์ Next.js
- GitHub Repository
- เว็บไซต์ออนไลน์บน Vercel
- แผนวัดผลเบื้องต้น
- Roadmap ว่าจะพัฒนาเว็บต่ออย่างไร

## วิธีใช้คู่มือนี้

1. อ่าน `full-student-handbook.md` เป็นเนื้อหาหลัก
2. ใช้ `03-prompt-pack-for-students.md` เวลาต้องคัดลอก Prompt ไปใช้กับ AI
3. กรอก `04-student-worksheet.md` ระหว่าง Workshop
4. ใช้ `05-final-submission-checklist.md` ตรวจงานก่อนส่งท้ายคอร์ส

ไม่ต้องข้ามไปทำโค้ดทันที เพราะเว็บที่ดีเริ่มจากโจทย์ที่ชัดก่อน แล้วค่อยใช้ AI ช่วยทำให้เร็วขึ้น
