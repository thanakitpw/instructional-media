# สิ่งที่ผู้เรียนต้องเตรียมก่อนเข้าเรียน
## Vibe Code Website Bootcamp

คอร์สนี้เป็นคอร์สลงมือทำจริง ผู้เรียนจะได้สร้างเว็บไซต์ด้วย AI ตั้งแต่การวางโจทย์ ออกแบบหน้าเว็บ เขียนเว็บด้วย Next.js และ Deploy ขึ้นออนไลน์ ดังนั้นควรเตรียมสิ่งต่อไปนี้ให้พร้อมก่อนวันเรียน

---

## 1. อุปกรณ์ที่ต้องใช้

ผู้เรียนควรเตรียม

- Notebook หรือ Computer ส่วนตัว
- อินเทอร์เน็ตที่เสถียร

แนะนำให้ใช้เครื่องที่สามารถติดตั้งโปรแกรมได้เอง เพราะระหว่างเรียนจะมีการติดตั้งเครื่องมือสำหรับเขียนเว็บและรันโปรเจกต์

---

## 2. บัญชีที่ควรสมัครไว้ล่วงหน้า

ผู้เรียนควรสมัครและ Login ให้เรียบร้อยก่อนวันเรียน

### บัญชีหลักที่จำเป็น

#### 1. Gmail

ใช้สมัครบริการต่าง ๆ เช่น GitHub, Vercel, Supabase และเครื่องมือ AI

#### 2. GitHub

ใช้เก็บโค้ดเว็บไซต์ และเชื่อมต่อกับ Vercel เพื่อ Deploy เว็บไซต์

#### 3. Vercel

ใช้สำหรับนำเว็บไซต์ Next.js ขึ้นออนไลน์

#### 4. Supabase

ใช้สำหรับทำ Database / เก็บข้อมูลจากฟอร์ม เช่น Lead Form หรือ Contact Form

#### 5. เครื่องมือ AI ที่จะใช้เรียน

เลือกใช้อย่างน้อย 1 ตัว เช่น

- ChatGPT
- Claude
- Cursor หรือ Agentic IDE ตัวอื่น
- Claude Code
- Codex

หมายเหตุ: บางเครื่องมืออาจมีค่าใช้จ่ายหรือมีข้อจำกัดในการใช้งาน ผู้เรียนควรตรวจสอบแพ็กเกจของตัวเองก่อนเข้าเรียน

---

## 3. โปรแกรมที่ควรติดตั้งก่อนเรียน

แนะนำให้ติดตั้งโปรแกรมเหล่านี้ไว้ก่อน

### จำเป็น

#### Google Chrome

ใช้เปิดเว็บไซต์ ทดสอบหน้าเว็บ และใช้งานเครื่องมือออนไลน์

#### VS Code หรือ Cursor หรือ Agentic IDE ตัวอื่น

ใช้เขียนและแก้ไขโปรเจกต์เว็บไซต์

#### Node.js LTS

ใช้รันโปรเจกต์ Next.js

#### Git

ใช้เชื่อมโปรเจกต์กับ GitHub

### แนะนำเพิ่มเติม

#### Postman หรือ Thunder Client

ใช้ทดสอบ API ถ้ามีช่วงสอนเรื่องฟอร์มหรือ Backend

#### Canva / Figma

ใช้ดูตัวอย่างดีไซน์ หรือเตรียม Reference เว็บไซต์

---

## 4. สิ่งที่ควรทดสอบก่อนวันเรียน

เพื่อให้เรียนได้ต่อเนื่อง ไม่เสียเวลาติดตั้งตอนเรียน ผู้เรียนควรลองเปิด Terminal หรือ Command Prompt แล้วเช็กคำสั่งเหล่านี้

```bash
node -v
npm -v
git --version
```

ถ้าคำสั่งเหล่านี้แสดงเลขเวอร์ชันขึ้นมา แปลว่าเครื่องพร้อมใช้งานเบื้องต้นแล้ว

ถ้าคำสั่งใดไม่ขึ้น หรือขึ้นว่า command not found แนะนำให้ติดตั้งโปรแกรมนั้นให้เรียบร้อยก่อนวันเรียน

---

## 5. ไอเดียเว็บไซต์ที่อยากทำในคอร์ส

ผู้เรียนควรเตรียมไอเดียเว็บไซต์มาอย่างน้อย 1 โปรเจกต์ เพราะในคอร์สจะใช้โปรเจกต์นี้ทำ Workshop ตั้งแต่ต้นจนจบ

ตัวอย่างเว็บไซต์ที่เลือกทำได้

- Landing Page ขายสินค้า / บริการ
- เว็บไซต์ธุรกิจ
- Portfolio ส่วนตัว
- เว็บแนะนำคอร์สเรียน
- เว็บรับ Lead / เก็บข้อมูลลูกค้า
- เว็บสำหรับยิงแอด
- เว็บร้านอาหาร
- เว็บคลินิก
- เว็บร้านเสริมสวย
- เว็บ Agency

แนะนำให้เลือกโปรเจกต์ที่ขอบเขตไม่ใหญ่เกินไป และสามารถทำเป็นหน้าเว็บหลัก 1 หน้าได้ก่อน

---

## 6. ข้อมูลธุรกิจหรือข้อมูลโปรเจกต์ที่ควรเตรียม

เพื่อให้ AI ช่วยทำเว็บได้แม่นขึ้น ผู้เรียนควรเตรียมข้อมูลเหล่านี้

- ชื่อธุรกิจ / ชื่อโปรเจกต์
- ธุรกิจหรือเว็บไซต์นี้ทำเกี่ยวกับอะไร
- กลุ่มเป้าหมายคือใคร
- จุดขายหลักคืออะไร
- ปัญหาของลูกค้าคืออะไร
- อยากให้คนเข้าเว็บทำอะไร เช่น ทัก LINE, กรอกฟอร์ม, สมัครเรียน, จองคิว
- บริการหรือสินค้าที่ต้องการนำเสนอ
- ราคา / แพ็กเกจ ถ้ามี
- คำถามที่ลูกค้ามักถามบ่อย
- ช่องทางติดต่อ เช่น LINE, Facebook, เบอร์โทร, Email

ถ้ายังไม่มีข้อมูลครบ สามารถเตรียมมาแบบคร่าว ๆ ได้ แล้วใช้ AI ช่วยจัดระเบียบในคลาส

---

## 7. ไฟล์และรูปภาพที่ควรเตรียม

ถ้ามีไฟล์เหล่านี้ จะช่วยให้เว็บที่ทำในคอร์สดูสมจริงมากขึ้น

- Logo
- รูปสินค้า / รูปบริการ
- รูปเจ้าของธุรกิจหรือทีมงาน
- รูปผลงานที่ผ่านมา
- Brand Color หรือโทนสีที่อยากใช้
- ตัวอย่างเว็บไซต์ที่ชอบ 2-3 เว็บ
- ข้อความแนะนำธุรกิจ
- รีวิวลูกค้า ถ้ามี
- Portfolio / Case Study ถ้ามี

ถ้ายังไม่มีรูปหรือโลโก้ สามารถใช้ Placeholder หรือให้ AI ช่วยวางโครงก่อนได้

---

## 8. สิ่งที่ควรเตรียมด้านความเข้าใจ

ผู้เรียนไม่จำเป็นต้องเขียนโค้ดเก่ง แต่ควรรู้พื้นฐานเหล่านี้คร่าว ๆ

- เว็บไซต์คืออะไร
- หน้าเว็บประกอบด้วย Section อะไรบ้าง
- Landing Page ใช้ทำอะไร
- CTA คืออะไร
- Form ใช้เก็บข้อมูลอะไร
- Domain / Hosting / Deploy คืออะไร
- AI ช่วยทำเว็บได้อย่างไร

คอร์สนี้จะเน้นให้เข้าใจ Workflow และลงมือทำจริง ไม่จำเป็นต้องจำโค้ดทั้งหมด

---

## 9. สิ่งที่ควรเตรียมใจ

ระหว่างเรียนอาจเจอ Error หรือเว็บไม่ขึ้นในบางขั้นตอน ซึ่งเป็นเรื่องปกติของการทำเว็บไซต์จริง

สิ่งที่ผู้เรียนจะได้ฝึกคือ

- อ่าน Error เบื้องต้น
- ใช้ AI ช่วย Debug
- แก้ไขโค้ดทีละจุด
- ปรับหน้าเว็บจาก Feedback
- Deploy และทดสอบเว็บจริง

เป้าหมายไม่ใช่แค่ทำเว็บให้เสร็จ แต่คือเข้าใจวิธีคิดและวิธีแก้ปัญหาเวลาใช้ AI ทำเว็บไซต์

---

# สรุปสั้น ๆ

ผู้เรียนควรเตรียมให้พร้อม 3 อย่างหลัก ๆ

```text
เครื่องมือพร้อม
+
บัญชีพร้อม
+
ไอเดียเว็บพร้อม
```

ถ้าเตรียมครบ จะสามารถตาม Workshop ได้ทัน และมีโอกาสทำเว็บไซต์ของตัวเองขึ้นออนไลน์ได้จริงภายในคอร์ส
